This project, developed during my Computer Science studies, demonstrates a complete end-to-end machine learning workflow. It covers everything from initial data exploration and preprocessing to model training using the Random Forest algorithm and final deployment as a web service via Flask.

## Project Overview
The goal of this project was to bridge the gap between a raw dataset and a functional, accessible API. By utilizing a Random Forest regressor/classifier, the project handles complex data relationships and provides a robust backend for real-time predictions.
